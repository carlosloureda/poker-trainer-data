# Roadmap: Próximas Bombas 💣🚀

## Fase 1: Identidad y Utilidad Inmediata
- [ ] Branding completo (**RangeCraft Studio**).
- [ ] Implementación de **Print Builder** (Exportación a PDF).

## Fase 2: El Entrenador (The Trainer)
- **Mesa de Juego**: Implementar una mesa física con jugadores, cartas y chips.
- **Modo Feedback**: Testear rangos en situaciones reales generadas aleatoriamente.
- **Sistema de Puntuación**: Evaluar la precisión de las decisiones preflop en base a los rangos definidos.

## Fase 3: Bulk Analysis (Data Pipeline)
- **Parser de Historiales**: Soporte inicial para **Winamax** (y posteriormente PokerStars, GGPoker).
- **Script de Preprocesado**: Analizar miles de manos automáticamente.
- **Validación de Rangos**: Comparar el juego real del usuario contra sus rangos de estudio y detectar fugas (leaks).

## Fase 4: Mega Análisis e IA
- **Análisis Postflop**: Integración de scripts avanzados para limpiar y categorizar manos.
- **IA Experimental**: Uso de LLMs/Modelos específicos para analizar contextos dinámicos postflop y ofrecer sugerencias estratégicas.
